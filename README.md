# ForpostAeroService_Website

Frontend-only corporate website for modern robotics company.

## Tech stack
Frontend: React (v18+)
Styling: Tailwind CSS
Animation: Framer Motion
Internationalization: i18next

## Installation & Setup

Because this project uses the latest libraries, it requires a specific installation flag to handle peer dependency versions.

Clone the repository:
git clone https://github.com/your-username/forpost-website.git
cd forpost-website

Install dependencies:
Note: Use the --legacy-peer-deps flag to avoid version conflicts between react-scripts and newer library versions.
npm install --legacy-peer-deps

Start the development server:
npm start

##Project Structure

src/components: Reusable UI elements (Header, Footer, Transitions).
src/pages: Main page views (Home, Drones, Service, Contact).
src/i18n.js: The "brain" of the translation system.
